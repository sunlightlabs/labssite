What this is

On December 8th the White House released the Open Government Directive requiring all federal agencies to release new datasets and to create (agencyname).gov/Open pages with their open government plans. We’ve set up this site to keep track of their success in meeting that deadline. This application checks each major agency website’s /Open page to see if it exists. If it does, a shiny green check mark goes next to it and a link to the /Open page comes up. If it doesn’t, then it gets an X by the name until it does.

This list is not all-encompassing of all the agencies that the Open Government Directive impacts. But it is a good start to see how the landscape will change.
